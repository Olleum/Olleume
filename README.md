💳 Проект 1: Family Budget MVP (Product Vision & Design)

Кейс по проектированию социально-финансового сервиса внутри банковской экосистемы. Фокус на UX-аналитике и приоритизации фич в условиях сжатых сроков.

📌 Intro

Задача — за 4 месяца спроектировать MVP сервиса «Семейный бюджет». Проект направлен на рост LTV и удержание клиентов через совместное управление финансами. В основе лежит решение проблемы прозрачности трат внутри семьи.

🧪 Аналитика и проектирование

User Story Mapping: Выстроена карта пользовательского пути для 3-х ролей (Родитель, Ребенок, Юзер).
Приоритизация: Из общего бэклога выделено 12 критических User Stories (категория Must) для запуска MVP.
UX-прототипирование: Разработаны Low-fidelity wireframes интерфейса настройки лимитов и уведомлений.

🔧 Strategy

Методология: Выбран Agile (Scrum) для возможности быстрой корректировки UX на основе фидбека.
Стек управления: Yandex Tracker для бэклога, Unidraw для маппинга, Yandex Wiki для документации.

📈 Результаты

Спроектирована логика лимитов с защитой от «точек отказа» (учет периодов, обратная связь).
Сформирован DoD (Definition of Done) для всех продуктовых задач.
